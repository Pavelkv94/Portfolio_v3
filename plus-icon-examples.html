CSS Plus Icon Examples

Method 1: Using pseudo-elements (::before and ::after)

Blue square plus icon
.plus-icon-1 { position: relative; width: 40px; height: 40px; background-color: #007bff; border-radius: 4px; } .plus-icon-1::before, .plus-icon-1::after { content: ''; position: absolute; background-color: white; } .plus-icon-1::before { width: 20px; height: 4px; top: 50%; left: 50%; transform: translate(-50%, -50%); } .plus-icon-1::after { width: 4px; height: 20px; top: 50%; left: 50%; transform: translate(-50%, -50%); }

Method 2: Circular plus icon

Green circular plus icon
.plus-icon-2 { width: 40px; height: 40px; background-color: #28a745; border-radius: 50%; position: relative; } .plus-icon-2::before { content: ''; position: absolute; top: 50%; left: 50%; width: 20px; height: 2px; background-color: white; transform: translate(-50%, -50%); } .plus-icon-2::after { content: ''; position: absolute; top: 50%; left: 50%; width: 2px; height: 20px; background-color: white; transform: translate(-50%, -50%); }

Method 3: Using box-shadow (alternative approach)

Red square plus icon with box-shadow
.plus-icon-3 { width: 40px; height: 40px; background-color: #dc3545; border-radius: 4px; position: relative; } .plus-icon-3::before { content: ''; position: absolute; top: 50%; left: 50%; width: 20px; height: 4px; background-color: white; transform: translate(-50%, -50%); box-shadow: 0 0 0 4px white; }

Method 4: Simple approach

Yellow square plus icon
.plus-icon-4 { width: 40px; height: 40px; background-color: #ffc107; border-radius: 4px; position: relative; } .plus-icon-4::before { content: ''; position: absolute; top: 50%; left: 50%; width: 20px; height: 4px; background-color: white; transform: translate(-50%, -50%); } .plus-icon-4::after { content: ''; position: absolute; top: 50%; left: 50%; width: 4px; height: 20px; background-color: white; transform: translate(-50%, -50%); }

Method 5: Animated plus icon

Purple animated plus icon (hover to see effect)
.plus-icon-5 { width: 40px; height: 40px; background-color: #6f42c1; border-radius: 50%; position: relative; cursor: pointer; transition: all 0.3s ease; } .plus-icon-5:hover { transform: scale(1.1); background-color: #5a32a3; } .plus-icon-5::before, .plus-icon-5::after { content: ''; position: absolute; background-color: white; transition: all 0.3s ease; } .plus-icon-5::before { width: 20px; height: 3px; top: 50%; left: 50%; transform: translate(-50%, -50%); } .plus-icon-5::after { width: 3px; height: 20px; top: 50%; left: 50%; transform: translate(-50%, -50%); }

Quick Summary

Key CSS properties used:

  • position: relative - for positioning context
  • ::before and ::after - pseudo-elements for the lines
  • content: '' - required for pseudo-elements
  • position: absolute - for precise positioning
  • transform: translate(-50%, -50%) - to center the lines
  • top: 50% and left: 50% - for centering
